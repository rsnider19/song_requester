## Overview
The landing screen when a confirmed performer is in Performer mode. Focused on managing their own gigs.

## Layout
- List of all their upcoming gigs
- Floating action button (FAB) to add a new gig

## Gig list items
- Same format as the audience home screen gig cards (performer name, venue, date, start/end time)
- Tapping a gig navigates to the gig screen
